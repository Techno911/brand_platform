# Канон «Бренд-платформа 3.1» (offline B2C, ЧиП)

Этот текст — **единственный источник истины** для methodology-compliance-check и всех судей.
Если в артефакте что-то противоречит канону — это нарушение независимо от вкуса критика.

## 0. Базовые принципы

- **Offline B2C**: методология применяется к бизнесам с физическим контактом (приём у врача, визит в салон, поход в ресторан, доставка мебели).
- **8-12 рабочих дней**: полный цикл от кик-оффа до финальной приёмки. Не 1 час, не 2 месяца.
- **Один собственник как источник правды**: ни фокус-группы, ни SEO-данные не перекрывают мнение собственника о миссии и ценностях. Задача методологии — **вытащить** мотив собственника, а не заменить его демоскопией.
- **Claude = черновик, человек = утверждение**. Никакое «зелёное» решение валидатора не попадает в финальный документ без явного клика маркетолога и, для ключевых артефактов, собственника.

## 1. Стадия 1. Voice of the Customer

Вход: доступ к отзывам, соцсетям, сайту, текущим продуктам.
Выход: 6 листов первичных данных (Sheet 1-6 Excel):
1. Интервью (CustDev ≥5 респондентов).
2. Отзывы (≥20 штук, любые источники).
3. Конкуренты (5-7 прямых + 2-3 косвенных).
4. Существующие обещания (сайт, соцсети, реклама).
5. Продукт (чек, география, штат).
6. Гипотезы сегментов (≤5 штук).

Железные правила:
- Каждая корзина pain/gain/JTBD имеет цитату-источник.
- Если ≤20 отзывов — статус `data_insufficient`, стадия 2 не запускается.
- Запрещено: додумывать боли, которых нет в источниках.

## 2. Стадия 2. Сессия с собственником

Вход: результаты Стадии 1.
Выход: легенда + 3-5 ценностей + 1 миссия + 1 видение.

### 2.1 Легенда
- ≥3 датированных факта в `milestones`.
- Основатель с именем и ролью.
- Ровно одна переломная точка (до/после).
- Никаких маркетинговых прилагательных.

### 2.2 Ценности
- Ровно 3-5 штук (1-2 — бренд не определим, 6+ — нечего помнить).
- 1-3 слова каждая, именительный падеж.
- ≥2 бытовых проявления на ценность.
- Обязательно `opposite` (что НЕ эта ценность).
- `source_quote` — из сессии или отзывов.

### 2.3 Миссия
- Без денежных маркеров (прибыль, доход, выручка, billion и т.п.).
- Без «лидерства в индустрии» (№1, доминировать).
- Активный залог.
- Глагол-действие (делать, возвращать, сохранять, давать).
- Конкретная целевая группа.
- ≤20 слов.

### 2.4 Видение
- Картина мира через 5-10 лет, если миссия выполнена.
- Не «мы станем №1» — а «люди в городе перестанут X».
- ≤30 слов.

## 3. Стадия 3. Позиционирование + архетип + месседж

### 3.1 Архетип
Ровно 1 `primary` из 12 канонических: Innocent, Everyman, Hero, Outlaw, Explorer, Creator, Ruler, Magician, Lover, Caregiver, Jester, Sage.
`secondary` допустим максимум один, при равных мотивах, с evidence.
`shadow` — теневая сторона, обязательна.
`evidence` — цитаты/факты в поддержку.

### 3.2 WE-пара
- 3-5 `we_are` признаков.
- 3-5 `we_are_not` признаков.
- Без `we_are_not` позиционирование не принимается.

### 3.3 Позиционирование
- ≤25 слов.
- Шаблон обязателен: «Для <сегмент>, который <ситуация/боль>, <бренд> — это <категория>, потому что <отличие>».
- `category` конкретная, не «качественный сервис».
- `competitor_diff` через наблюдаемое действие, не через прилагательные.

### 3.4 Бренд-месседж
- **4-7 слов** включительно. 3 или меньше — warning. 8+ — отклонено.
- Без цифр и чисел.
- Без денежных маркеров.
- Без сравнительной степени без указания с кем.
- Без двойной пунктуации.
- Разрешена метафора с одной коннотацией.

## 4. Стадия 4. 4 теста

Ровно 4 параллельных теста. Pass = >50% по каждому:
1. **Memorability**: через 20 минут отвлечённой беседы 3+ человека из 5 повторяют точно.
2. **Differentiation**: из 3+ месседжей конкурентов тестируемый выделяется без подсказок.
3. **Claim-backing**: каждое утверждение подтверждается фактом из легенды или продукта.
4. **Emotional-hook**: 3+ из 5 человек показывают эмоциональную реакцию (улыбка, удивление, печаль).

Если любой тест fail — месседж не утверждается, возврат на Стадию 3.

## 5. Анти-паттерны (блокируются на всех стадиях)

- Маркетинговые клише: «качество», «клиентоориентированность», «инновации», «премиум», «№1», «лидер».
- Позиционирование через сравнительную степень без конкретной цифры.
- Архетип без evidence.
- Миссия с денежными маркерами.
- Легенда без датированных фактов.
- Ценности без proof-of-life (manifestations).
